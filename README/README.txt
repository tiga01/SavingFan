---------------------
MCU为:STM32F103C8T6
系统上电自动开启测温，温度阈值默认为23℃
KEY1、KEY2为增、减键，KEY3按键为功能键
可调节风扇工作模式：自动模式、风（手）动模式
通过USB-A连接到PC端之后，打开串口助手软件，自动向串口助手发送风扇工作状态、环境温度、阈值温度
OLED屏幕实时显示风扇工作状态、环境温度、阈值温度
通过ESP8266-01S模块，可与云平台连接，进行无线通讯
	与巴法云连接可实现云端控制，在云端输入相关指令，即可控制风扇、RGB灯、蜂鸣器的或设置温度阈值、风扇转速等......
	与阿里云连接可实现云端实时监控，可看到当前环境温度、风扇转速的曲线图
	以上两种云平台均可生成移动端软件
系统在正常运行时会有蓝色灯光闪烁
---------------------

---------------------
自动模式下：
	环境温度小于阈值温度，风扇不工作
	环境温度大于阈值温度：风扇转速（PWM波空占比）随温度增高而增高，在30℃时达到最高
	按键可调节温度阈值，可调范围在20℃~30℃
-------------------

-------------------
风（手）动模式下：
	通过按键手动调节风扇转速（PWM波空占比）
